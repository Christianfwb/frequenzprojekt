## 8.0 Die ewige Suche nach dem Warum

Die modernen Formeln der Physik beschreiben das **Wie** der Welt mit
außerordentlicher Präzision. Das **Warum** bleibt als offene ontologische Frage.

> Das Frequenzgesetz schlägt eine alternative ontologische Lesart vor —
> ohne eine einzige Formel zu verändern.

## 8.1 Die Lichtgeschwindigkeit c

```
c = λ · f
```

c beschreibt die maximale Ausbreitungsgeschwindigkeit physikalischer
Wechselwirkungen im Vakuum. Im Frequenzgesetz ist dies eine strukturelle
Eigenschaft der Ausbreitung von Frequenzmustern (→ Kapitel 06.2).

---

## 8.2 Die Planck-Formel E = hf

```
E = h · f   (→ Axiom A4)
```

Max Planck führte 1900 die Quantisierung als mathematische Notwendigkeit ein.
Warum Energie portioniert sein sollte, blieb konzeptuell offen.

Im Frequenzgesetz: Die Quantisierung kann als Konsequenz diskreter stabiler
Frequenzzustände interpretiert werden — Frequenz ist nicht Wirkung der Energie,
sondern ihre Ursache.

---

## 8.3 Einsteins Masseformel E = mc²

```
m = h · f / c²   (→ Axiom A5)
```

Einstein bewies die Austauschbarkeit von Masse und Energie. Was Masse
ontologisch ist, blieb offen.

Im Frequenzgesetz: Masse ist stabilisierte, gebundene Frequenzstruktur.
Trägheit ist der Widerstand einer stabilen Frequenzkonfiguration
gegen Zustandsänderung.

---

## 8.4 Heisenbergs Unschärfe Δx·Δp ≥ ℏ/2

Ein Teilchen ist ein Wellenpaket — eine Fourier-Summe von Frequenzen:

```
Schärfer im Raum (Δx klein)  →  breiter im Impuls (Δp groß)
```

Das ist eine mathematische Eigenschaft der Fourier-Darstellung, keine
mysteriöse Messgrenze. Die Unschärfe ist eine unvermeidliche Konsequenz
der Lokalisierung eines Wellenpakets.

---

## 8.5 Die Schrödinger-Gleichung HΨ = EΨ

```
HΨ = EΨ
```

Standardinterpretation: Ψ ist eine Wahrscheinlichkeitsamplitude.

Im Frequenzgesetz: Ψ beschreibt alle möglichen Frequenzzustände eines Systems.
Der Messprozess stabilisiert einen bestimmten Frequenzmodus durch Wechselwirkung
mit dem Messapparat.

```
|Ψ|² → Verteilung möglicher stabiler Frequenzzustände
```

*Diese Interpretation ist nicht experimentell von der Standardinterpretation
unterscheidbar — sie ist eine ontologische Lesart, keine neue Physik.*

## 8.6 Bose-Einstein-Kondensat

```
BEC: alle Teilchen → selber makroskopischer Quantenzustand
   → maximale Phasensynchronisation
```

Das BEC kann als reinste experimentell zugängliche Form von Frequenzkohärenz
interpretiert werden (→ Kapitel 04.4). Alle Teilchen teilen denselben
quantenmechanischen Zustand — maximale Phasensynchronisation.

## 8.7 Die Gravitationskonstante G

G beschreibt die Kopplungsstärke gravitativer Wechselwirkungen.
Im Frequenzgesetz kann G als Maß für die Stärke der Phasenkopplung
zwischen Frequenzknoten gelesen werden.

Kapitel 09 entwickelt Gravitation als Phasensynchronisation vollständig.

## 8.8 Das große Muster

| Formel | Klassisch gelesen | Frequenzgesetz-Lesart |
|---|---|---|
| E = hf | Energie bestimmt Frequenz | Frequenz primär, Energie abgeleitet (A4) |
| E = mc² | Masse ist Energie | Masse als gebundene Frequenz (A5) |
| T = 1/f | Zeit ist Periodendauer | Spezialfall: T = ΔΦ/f bei ΔΦ = 2π (A3) |
| HΨ = EΨ | Wahrscheinlichkeitskollaps | Resonanzauswahl im Frequenzfeld |
| Δx·Δp ≥ ℏ/2 | Messgrenze | Fourier-Eigenschaft von Wellenpaketen |
| G | Anziehungskraft | Phasensynchronisation (→ Kap. 09) |

Alle Formeln bleiben mathematisch identisch.
Was sich ändert, ist ausschließlich die Leserichtung ihrer kausalen Ordnung.
